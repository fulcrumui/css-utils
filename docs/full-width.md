# Full Width Module

Applies a width of 100% to the element that it's applied to

## Usage

### Included in core: ✅

### Application

To apply this utility, use the pattern below:

```html
<element class="u-full-width"></element>
```

### Options

| Option | Description |
| --- | --- |
| **N/A** | There are no options for this module |
